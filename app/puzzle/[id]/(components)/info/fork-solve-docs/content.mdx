### Solve locally (WIP)

1. Clone [GitHub repo](https://github.com/waterfall-mkt/curta-puzzles) + install deps
```sh showLineNumbers={false} containerized={false}
git clone https://github.com/waterfall-mkt/curta-puzzles.git && cd curta-puzzles && forge install
```

2. Set `RPC_URL_MAINNET` in `.env`
```sh fileName=".env" showLineNumbers={false} containerized={false}
RPC_URL_MAINNET=""
```

3. Write solution + run script
```sh showLineNumbers={false} containerized={false}
forge script <PATH_TO_PUZZLE> -f mainnet -vvv
```
